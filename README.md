# This is a Swiggy-style food delivery analytics dashboard designed to track sales performance, customer behavior, and regional trends. Let’s go through it in a structured, easy-to-understand way.

# 1. Left Sidebar – Navigation & Filters
🔹 Navigation Buttons
Dashboard / Data / Analysis
Likely used to switch between different report pages.
🔹 Filters (Interactive Controls)
Month Filter → January, February
State Filter → Delhi, Karnataka, Maharashtra
Restaurant Filter → Specific restaurant names

👉 Purpose:
These filters dynamically update all visuals, allowing deep analysis like:

“Sales in Karnataka during February”
“Performance of a specific restaurant”

# 3. Monthly Trends (Line Chart)
Compares performance across months (January vs February)
Observations:
January shows higher sales (~3.5M) than February (~2.1M)
Indicates a decline in February

👉 Possible reasons:

Seasonal demand drop
Fewer promotions
External factors (weather, holidays)

# 4. Daily Trends (Bar Chart)
Shows day-by-day sales performance
Insights:
Fluctuating daily sales pattern
Some peaks (~0.3M) → high-demand days
Some dips (~0.05M) → low activity

👉 Useful for:

Identifying best-performing days
Planning offers or staffing
